# Dotfiles

Personal configs managed with [GNU Stow](https://www.gnu.org/software/stow/).

## Install

```bash
./install.sh
```

This symlinks all configs into `$HOME` using Stow.

## Configs

- [bash](bash/README.md) — fallback shell config
- [git](git/README.md) — sensible git defaults
- [nushell](nushell/README.md) — primary shell with vi mode and fuzzy completions
- [nvim](nvim/README.md) — Neovim 0.11+ with LSP, Telescope, and format-on-save
- [starship](starship/README.md) — cross-shell prompt
